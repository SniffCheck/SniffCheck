SniffCheck Web Flasher

RF environment audit tool for the LilyGO T-Dongle C5.

Flash SniffCheck onto a T-Dongle C5 straight from your browser. No ESP-IDF, no Python, no drivers. Plug the dongle into a USB port, click Install, pick the serial port, and wait.

This browser can't flash over USB. Use Chrome or Edge on a desktop (Windows, macOS, or Linux). Firefox, Safari, and all phones/tablets lack the Web Serial API.

If the dongle isn't detected or flashing fails: unplug it, hold the BOOT button, plug it back in while still holding, then release and click Install again. This forces download mode.

What it does

SniffCheck passively audits the Wi-Fi (both bands) and BLE around you and shows a safety verdict on its little screen. It only listens — it never connects to or transmits to other devices, and nothing leaves the dongle. The flash includes the firmware and the vendor identification database, so it scans correctly on first boot.

Requirements

  • Chrome or Edge on desktop (Web Serial API required).
  • A LilyGO T-Dongle C5 (ESP32-C5).
  • A USB port. Flashing erases the dongle and installs a clean image.
